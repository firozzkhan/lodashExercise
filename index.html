Array Methods

Exercise 1: _.flatten()

                
                    Flattens array a single level deep.
                    example:
                    var result = _.flatten([1, 2, [4, 3, [6, 7], 3, 2], 9, 8]);

Exercise 2: _.head()

                
                    Gets first Element.
                    example:
                    var result = _.head([1,2,[4,3,[6,7],3,2],9,8]);

Exercise 3: _.indexOf()

                
                    Gets index of First Occurence of Element.
                    example:
                    var result = _.indexOf(_.flatten([1, 2, [4, 3, [6, 7], 3, 2], 9, 8]), 4);

Exercise 4: _.initial()

                
                    Give array excluding last element.   
                    example:
                    var result = _.initial([1,2,[4,3,[6,7],3,2],9,8]);

Exercise 5: _.last()

                
                    Give last element of array.
                    example:
                    var result = _.last([1, 2, [4, 3, [6, 7], 3, 2], 9, 8]);

Exercise 6: _.nth()

                
                    Give element at nth index. If n is -ve then nth element from end is returned.

Exercise 7: _.pull()

                
                    Returns array by removing  given values.
                    _.pull(array, values)

Exercise 8: _.pullAll()

                
                    Returns array by removing  given values in array.
                    example: _.pullAll(array, [array of values to be removed])

Exercise 9: _.remove

                
                    Gives array of removed values if function returns true.
                    _.remove(array, function);
